Install

  • NodeJS
  • MongoDB
  • Install Yeoman & Beardme
    npm install -g yeoman generator-beardme grunt-cli
yo beardme
grunt init
grunt test
grunt server

Server-Side Components

  • Express - HTTP/API
  • Mongoose - MongoDB
  • Winston - Logging
  • Socket.IO - Websockets

Browser-Side Components

  • RequireJS - Loader
  • Backbone - MVC Framework
  • Bootstrap - The Pretty
