#Beard Me
Based on Randy Lebeau's Stacked, but customized to suit my tastes.
##Install
- [NodeJS](http://nodejs.org/)
- [MongoDB](http://www.mongodb.org/downloads)
- Install Yeoman & Beard Me
```javascript
npm install -g yeoman generator-beardme grunt
yo beardme
grunt init
grunt server
```
